# 🎫 Digital Pass Management System

A web-based Digital Pass Management System developed to streamline the process of requesting, managing, and approving digital passes. The application provides role-based access for administrators, officers, and users, ensuring efficient pass management through a centralized database.

---

## 📌 Features

- User registration and login
- Role-based authentication
- Admin dashboard
- Officer dashboard
- Visitor dashboard
- Pass request management
- Database-driven record management
- Secure storage of user and pass information

---

## 🛠️ Tech Stack

| Category | Technology |
|----------|------------|
| Backend | Python, Flask |
| Frontend | HTML, CSS |
| Database | SQLite |
| Development | VS Code |

---

## 📂 Project Workflow

1. Users register and log in to the system.
2. Users submit pass requests.
3. Officers review and process requests.
4. Administrators manage users and system records.
5. Pass information is stored and managed using SQLite.
